# SweetTreats Bakery Website

## Overview
This is the **SweetTreats Bakery website** — a free, fully functional online bakery site designed for GitHub Pages.  
It allows customers to order cakes, cupcakes, cookies, churros, cake pops, and a special treat.  

### Features
- Poster-style professional design  
- Editable product images through the parent panel  
- Custom cake ordering: choose size, flavor, shape, and name on the cake  
- Delivery orders only (no pickup)  
- Total price calculation based on size and flavor  
- WhatsApp order integration  
- Parent panel (password `2459`) for:  
  - Updating prices  
  - Updating product images  
  - Confirming/resetting order status  
- Editable Till Number for payment  

## How to Use
1. Upload `index.html` (and optional images) to GitHub Pages or any hosting.  
2. Open the website in a browser.  
3. Customers can place orders using the order form.  
4. Parent/admin can access the panel via the **Parent Panel Access** button and the password `2459`.  
5. Update prices, images, or confirm payment directly in the panel.  

## Customization
- Change product images by pasting new URLs in the parent panel.  
- Update cake prices or extra flavor costs in the parent panel.  
- Edit Till Number for payments using the panel.  

## Hosting
- Recommended free hosting: [GitHub Pages](https://pages.github.com)  
- Steps: Create a repository → Upload `index.html` → Enable GitHub Pages (main branch, root folder) → Visit live URL.
